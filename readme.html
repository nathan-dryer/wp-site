# NateDryer.com WordPress Site ## Description This is the repository for Nate Dryer's personal WordPress site, using the Elementor Pro plugin for page creation. The site is designed to be fast, efficient, and easy to maintain. ## Requirements - WordPress 5.8 or higher - PHP 7.4 or higher - MySQL 5.6 or higher - Plugins: - Elementor - Elementor Pro (requires a valid license) - Free Customer Service Tools by OpenWidget - LiteSpeed Cache
